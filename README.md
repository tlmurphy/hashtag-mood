# #Mood

Install pip packages:
```
pip install -r requirements
```

Create a .twitter-config file with keys:
```
consumer_key = ''
consumer_secret = ''
access_key = ''
access_secret = ''
```

or

Place keys in env variables:
```
CON_KEY
CON_SECRET
ACC_KEY
ACC_SECRET
```

Serve app:
```
python app.py
```
